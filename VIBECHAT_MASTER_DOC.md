## 3. Key Features & Superpowers

VibeChat is built on six core pillars of functionality, designed to empower every member of the group.

### üß† 1. The Architect (Custom AI Personalities)

* **Function:** Every chat comes with a built-in AI member (powered by GPT-5) that lives in the group.
* **Customization:** The group creator can "code" the AI's personality.
  * **Name:** Give it a name (e.g., "Jarvis", "The Dungeon Master", "Roasted Bot").
  * **Personality Prompt:** Define how it acts (e.g., "You are a ruthless 1980s stockbroker," "You are a supportive therapist," "You are a chaotic hype-man").
  * **Tone:** Select from presets like Professional, Chill, Unhinged, or Sarcastic.
* **Capability:** The AI "sees" images (via GPT-Vision), remembers context (last 100 messages), and participates naturally without needing to be pinged constantly (via Smart Engagement).
* **Tools:** Equipped with **Web Search** (for real-time info) and **Image Generation** (for visual expression), allowing it to fetch live data or create art on demand.

### üõ†Ô∏è 2. The Engineer (Custom Slash Commands)

* **Function:** Users can build their own chat tools using natural language prompts.
* **How it Works:** Define a trigger (e.g., `/roast`) and an instruction (e.g., "Insult the user based on their last message").
* **Multi-Tool Power:** Commands can intelligently invoke:
  * **Web Search:** `/factcheck` (Search Google for the truth).
  * **Image Gen:** `/visualize` (Generate DALL-E 3 art).
  * **Data Analysis:** `/analyze` (Parse CSVs or text).
* **Examples:**
  * `/bet` -> "Check the odds for the game mentioned."
  * `/translate` -> "Translate to Gen Z slang."
  * `/dnd_roll` -> "Roll a d20 and describe the outcome dramatically."

### üé¨ 3. The Director (Smart Threads)

* **Function:** Intelligent noise control.
* **The Problem:** Important plans get lost in memes.
* **The Solution:** Create **Smart Threads** that filter the main chat view.
  * **Logic Gates:** Filter by Topic ("Gaming", "Work"), Keywords ("Dinner", "Time"), or People.
  * **Unified Timeline:** Messages sent in threads still appear in the main chat (no hidden pockets), but you can *view* the chat through a filtered lens to see only what matters.
  * **Use Case:** Switch to the "Plans" thread to see only logistics, ignoring the 500 messages of banter in between.

## 6. Technical Overview

Built for speed, privacy, and scalability.

* **Frontend:** React Native (Expo SDK 53) with NativeWind (Tailwind).
  * *Why:* Native performance (60fps), beautiful gestures, cross-platform.
* **Backend:** Bun + Hono (High-performance JS runtime).
  * *Why:* Ultra-low latency API responses.
* **Database:** Prisma + SQLite (Scaling to Postgres).
  * *Why:* Robust data integrity and fast queries.
* **AI Engine:**
  * **Orchestration:** OpenAI Responses API (GPT-5).
  * **Vision:** GPT-4o-mini / GPT-5-vision.
  * **Image Gen:** DALL-E 3 & Google Gemini (Nano-Banana).
  * **Search:** Google Custom Search API.
